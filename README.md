# my-web-index

A simple static starter project scaffolded by GitHub Copilot.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Run
1. Open `index.html` in a browser.
2. Click the `Click me` button to display a message.

## Notes
No build step is required. Just open `index.html` directly or use a static file server.
